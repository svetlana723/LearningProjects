# Мои учебные проекты

Здесь собраны проекты, которые я делаю в процессе изучения программирования.

## Структура
- BrickGame2 — реализация игры «Змейка» на языке программирования С++. Разработанную библиотеку также можно подключить к консольному интерфейсу из BrickGame v1.0. Разработанную в BrickGame v1.0 игру «Тетрис» можно подключить к десктопному интерфейсу, разработанному в данном проекте.
- PythonProject — учебный проект на Python. 1 часть - Моделирование сдачи экзамена студентами. Каждый экзаменатор принимает экзамен в отдельном процессе. 2 часть - Скачивание изображений. Обработчик ссылок, который просит пользователя ввести ссылку на изображение, а затем асинхронно скачивает его.

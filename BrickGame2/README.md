# Змейка
Реализация игры «Змейка» на языке программирования С++.
Проект состоит из двух отдельных компонентов: библиотеки, отвечающей за реализацию логики игры, и десктопного графического интерфейса.

Разработанную библиотеку также можно подключить к консольному интерфейсу из BrickGame v1.0.
Разработанную в BrickGame v1.0 игру «Тетрис» можно подключить к десктопному интерфейсу, разработанному в данном проекте.
